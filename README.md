Diabetes Risk Classification & Patient Segmentation
Institution: Belgium Campus ITversity - Year 3 (Module MLG382)

Methodology: CRISP-DM Framework

This project utilizes machine learning to predict diabetes risk stages and segment patients based on health and lifestyle characteristics. It includes three classification models (Decision Tree, Random Forest, XGBoost) and an unsupervised clustering model (K-Means).


1. Prerequisites
Ensure you have Python 3.8+ installed. You will need the following libraries:
pip install xgboost shap joblib scikit-learn pandas
